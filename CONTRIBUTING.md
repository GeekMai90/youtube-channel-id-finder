# Contributing

## Development
1. Clone the repository.
2. Open `chrome://extensions/`.
3. Enable Developer mode.
4. Click `Load unpacked` and choose this project directory.
5. Edit files and click `Reload` on the extension card.

## Pull Requests
1. Create a branch from `main`.
2. Keep changes focused and include clear commit messages.
3. Test on YouTube channel pages before opening a PR.
4. Update `README.md` if behavior changes.

## Reporting Issues
Please include:
- Chrome version
- Example YouTube URL
- Expected behavior and actual behavior
- Console errors (if any)
